Phase 2
Read path
~120 BPM TBT reads → ring trajectory plot. CMD:ACQUIRE.
→ M1 validated · M2 next

Scope decisions locked

  • Read-only (no arming, no triggering) — phase-4 black box avoided
  • All ~120 BPMs, first turn, snapshot mode (no continuous monitoring)
  • Faithful MATLAB port for injection-turn detection (per-BPM argmax, 1370 fallback)
  • Result PVs (small extracted) + raw waveforms via REST
  • Best-effort with NaN holes; state PVs report failed counts/names
  • Persistent CA connections, on-demand parallel reads
  • UI layout: stacked X above Y (matches original MATLAB GUI)
